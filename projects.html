Travel Planner

We worded in a 3-person team, we created A full-stack application for planning, managing, and viewing travel trips with integrated Google APIs and database management

Tools

Backend : nodeJS, Express
Frontend : React with Vite, Bootstrap

Cloud platform : Heroku
Database : AWS DynamoDB
Storage : AWS S3

Code hosting platform : Github
Others : Trello

We called the Google Geocoding API to get the geographic code of the specific location, and then we called the Place Autocomplete method in Google Places API to get the place information. We also used the Google Maps API to display the map.

Lyrics Sercher

As a Spotify user, I found it inconvenient to search for lyrics while listening to music. So I created a web application that allows users to search for lyrics by song name and artist name. The application also provides a list in user's Spotify account of the songs you have listened to recently. And allow users to search for lyrics by clicking on the song.

Tools

Backend: Python FastAPI
Frontend: Html Pyscript
Cloud platform : Heroku
Database : PostgreSQL by Heroku

Pyscript is a python library that allows you to write HTML in python code. I used this library to create a web application that allows users to search for lyrics by song name, album name and artist name Experimentally.
